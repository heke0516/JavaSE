MyBatis 介绍
MyBatis 是一款优秀的持久层框架，它通过简单的 XML 或注解配置，将 Java 对象与数据库操作进行映射，简化了数据库访问层的开发。

核心特点
1. 半自动化的 ORM 框架
与 Hibernate 等全自动 ORM 不同，MyBatis 不自动生成 SQL

开发者需要手动编写 SQL，但结果映射自动完成

保留了 SQL 的灵活性，便于优化和复杂查询

2. SQL 与代码分离
SQL 语句写在 XML 文件或注解中，与 Java 代码解耦

便于维护和查看，支持动态 SQL
